# Summing two Random Numbers between 1-100
A simple program that generates to random numbers between 1 and 100. Presents these numbers in the command line and asks for the sum of the two numbers. When the correct answer is enters the message:
>Yes! \<answer> is the correct answer.   
>You got it in \<number of attempts> attempts.

will print. If a wrong answer is submitted the message:
>Wrong! The correct answer is greater than \<answer>.  

or  
>Wrong! The correct answer is less than \<answer>.

and the attempts counter will be incremented.
